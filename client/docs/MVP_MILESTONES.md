## План релизов MVP

*   Фаза 1: Singleplayer Core
    *   Создан GameLoop.
    *   Герой перемещается по 2D карте, реализован базовый рендер в Canvas.
    *   Работает ближний бой.
*   Фаза 2: Environment & Progression
    *   Реализована смена времени суток.
    *   Добавлен спавн ресурсов и система инвентаря.
    *   Добавлена механика голода и восстановления голода ягодами.
    *   Добавлены мобы для PvE.
*   Фаза 3: Game Systems
    *   Реализован AI мобов: Idle, Patrol, Chase, Attack, Flee.
    *   Добавлены базовые коллизии игрока, мобов и объектов карты.
    *   Добавлен подсчет статистики партии: счет, время выживания, убитые мобы, собранные ресурсы, дни.
*   Фаза 4: Backend & Polish
    *   Реализована авторизация через JWT.
    *   Реализована отправка результата партии на сервер.
    *   Реализован экран лидерборда с топом игроков и личной статистикой.
    *   Добавлены автоматизированные тесты для клиентской игровой логики и backend API.
