## Стек технологий и инструментарий

| Категория | Технология | Обоснование выбора |
| :--- | :--- | :--- |
| Язык | Kotlin | Нативный язык Android, поддержка Flow/Coroutines, строгая типизация. |
| Отрисовка и UI | Jetpack Compose | Декларативный UI для меню, HUD и экранов приложения. Использование Canvas API для рендера игровых примитивов и спрайтов. |
| Игровой цикл | Kotlin Coroutines | Организация игрового цикла и периодического обновления состояния. |
| Состояние | StateFlow | Передача актуального `GameState` из ViewModel в Compose UI. |
| Сеть | Retrofit & OkHttp | REST-интеграция клиента с backend API для авторизации и лидерборда. |
| Хранение сессии | SharedPreferences | Сохранение access/refresh token и данных пользователя на клиенте. |
| Backend | Spring Boot | REST API для авторизации и лидерборда. |
| База данных backend | PostgreSQL | Хранение пользователей и результатов партий. |
| Авторизация | JWT | Access/refresh token flow для защищенных запросов. |
| Клиентские тесты | JUnit | Unit-тестирование игровой логики, расчета очков и анимаций. |
| Android instrumented-тесты | AndroidX Test | Проверка приложения в Android-среде на устройстве или эмуляторе. |
| Серверные тесты | JUnit 5, Spring Boot Test, MockMvc | Проверка backend API, авторизации, лидерборда и серверного расчета очков. |
