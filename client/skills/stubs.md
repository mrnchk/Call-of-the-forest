# Навык: Заглушки (Stubs)

## Суть

Заглушки для фич, которые появятся позже. Пока — минимальный код, чтобы проект компилировался и работал.

---

## 1. AI мобов — ЗАГЛУШКА

Мобы есть в GameState, но стоят на месте. AI-система не подключена.

```kotlin
// Сейчас в GameEngine: мобы не двигаются
data class Mob(
    val id: String,
    val x: Float,
    val y: Float,
    val radius: Float = 14f,
    val hp: Int = 50,
    val type: MobType = MobType.IDLE
)

enum class MobType { IDLE }  // только один тип — стоит
```

**Когда добавлять:** Фаза 2. Нужен StateMachine (Idle → Chase → Attack → Flee).
**Что менять:** Добавить `AIComponent`, написать `aiSystem()`, подключить в конвейер GameLoop.

---

## 2. Боевая система — ЗАГЛУШКА

Нет урона, нет атак. `hp` есть в Player и Mob, но никто его не меняет.

```kotlin
// Player.hp = 100 — никогда не меняется
// Mob.hp = 50 — никогда не меняется
```

**Когда добавлять:** Фаза 2. Нужен AttackAction, хитбоксы, knockback.
**Что менять:** Добавить кнопку атаки, `combatSystem()`, подключить в конвейер.

---

## 3. Мультиплеер — ЗАГЛУШКА

Singleplayer only. Нет сервера, нет клиента.

**Когда добавлять:** Фаза 3. Ktor WebSockets.
**Что менять:** GameEngine на хосте = авторитетный сервер. На клиенте — рендер + отправка ввода.

---

## 4. Room сохранение — ЗАГЛУШКА

Состояние живёт только в памяти. Закрыл приложение — всё пропало.

**Когда добавлять:** Фаза 4. Room database.
**Что менять:** SaveManager с автосохранением GameState → Room.

---

## 5. Инвентарь — ЗАГЛУШКА

Нет предметов, нет крафта. `InventoryComponent` не добавлен в Entity.

**Когда добавлять:** Фаза 2.
**Что менять:** Добавить `InventoryComponent`, `Item` data class, `inventorySystem()`.
