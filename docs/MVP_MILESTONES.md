## План релизов MVP

*   Фаза 1: Singleplayer Core
    *   Создан GameLoop.
    *   Герой перемещается по 2D карте, реализован базовый рендер в Canvas.
    *   Работает ближний бой по статичным целям.
*   Фаза 2: Environment & Progression
    *   Смена времени суток (графическая).
    *   Добавлен спавн ресурсов, система инвентаря.
    *   Внедрены простые мобы для PvE.
*   Фаза 3: Multiplayer & Events
    *   Подключение второго игрока по сети, синхронизация позиций.
    *   Внедрение глобальных событий (гроза, вулкан, кровавая луна).
    *   Интеграция Алтаря и механики голода.
*   Фаза 4: Polish
    *   Лидерборды, сохранение/загрузка прогресса через Room.
    *   Оптимизация рендера и сетевого трафика (интерполяция).
