## Стек технологий и инструментарий

| Категория | Технология | Обоснование выбора |
| :--- | :--- | :--- |
| Язык | Kotlin | Нативный язык Android, поддержка Flow/Coroutines, строгая типизация. |
| Отрисовка и UI | Jetpack Compose | Декларативный UI для меню/инвентаря. Использование Canvas API для высокоскоростного рендера игровых примитивов и спрайтов без накладных расходов традиционных View. |
| Физика | JBox2D / dyn4j | Легковесные физические 2D движки для Android. Берут на себя AABB коллизии, расчет векторов и физику отбрасывания (knockback). |
| Асинхронность | Kotlin Coroutines & Flow | Организация игрового цикла, подписка на обновления состояния. |
| Сеть | Ktor Client & WebSockets | Высокопроизводительный асинхронный клиент для передачи бинарных данных/JSON в реальном времени. |
| БД (Локальная) | Room | Кэширование данных пользователя, сохранение сессии и инвентаря. |
| **Dependency Injection**| Hilt | Внедрение зависимостей для дао, репозиториев и сетевых клиентов. |
